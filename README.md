# INN Checker

Утилита для получения ИНН физического лица через сервис ФНС России.

> ⚖️ **Юридическое предупреждение:** Данный проект предназначен исключительно для законного использования при наличии письменного согласия субъекта персональных данных в соответствии с Федеральным законом РФ №152-ФЗ «О персональных данных» и внутренних политик безопасности. Автор не несёт ответственности за неправомерное применение кода.

## Возможности

- Запрос ИНН по ФИО, дате рождения и данным паспорта РФ
- Автоматическая инициализация сессии с сервисом ФНС
- Корректное форматирование серии и номера паспорта
- Механизм повторных запросов с нарастающей задержкой
- Цветное логирование с информативными сообщениями
- Обработка CAPTCHA и ошибок валидации

## Использование

### Быстрый запуск

```bash
python inn_checker.py
```

### Использование в качестве модуля

```python
from inn_checker import NalogInnClient

client = NalogInnClient(timeout=30, retries=3, retry_delay=2.0)

inn = client.get_inn(
    last_name="Иванов",
    first_name="Иван",
    patronymic="Иванович",
    birth_date="01.01.1990",
    passport_series="4510",
    passport_number="123456",
    passport_issue_date="15.06.2015"
)

if inn:
    print(f"ИНН: {inn}")
else:
    print("ИНН не найден")
```

## Формат данных

| Поле | Формат | Пример |
|------|--------|--------|
| Дата рождения | `ДД.ММ.ГГГГ` | `13.02.1991` |
| Серия паспорта | 4 цифры | `4510` |
| Номер паспорта | 6 цифр | `123456` |
| Дата выдачи паспорта | `ДД.ММ.ГГГГ` | `15.06.2015` |

Серию и номер паспорта можно передавать в любом виде — скрипт автоматически извлечёт цифры и отформатирует их в формат `XX XX XXXXXX`.

## Ограничения

- Поддерживается только **паспорт РФ** (код документа `21`)
- При запросе **CAPTCHA** автоматическое получение ИНН невозможно
- Сервис ФНС может ограничивать частоту запросов
